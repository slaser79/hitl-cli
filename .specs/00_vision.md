# Product Vision: HITL CLI

## Executive Summary
Open-source command-line interface and Python SDK serving as the reference implementation for Human-in-the-Loop (HITL) services. Enables programmatic requests for human input and notifications via mobile devices.

## Core Value Proposition
1. **Zero-Config:** Dynamic OAuth client registration - no manual setup required.
2. **Secure by Default:** Modern OAuth 2.1 with PKCE + optional End-to-End Encryption (E2EE).
3. **Dual-Mode:** Works as standalone CLI or as MCP proxy for Claude/Gemini/Codex integration.
4. **Autonomy:** Designed to be managed by AI Agents via the HITL Empire governance model.
